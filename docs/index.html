Dissolution Apparatus Hydrodynamics Explorer

Interactive tool for exploring validated CFD and PIV hydrodynamics data in USP Dissolution Apparatus 1 (basket type). Built on 5 peer-reviewed publications.

View on GitHub

What This Tool Does

Explore turbulence model selection, mesh independence, CFD–PIV validation, flow rates, power numbers, and mixing times — all in your browser. No CFD software required.

For Scientists & Engineers

Practical pharma applications: method development, OOS troubleshooting, regulatory justification with validated hydrodynamic data.

Turbulence Model Selection

Compare 4 RANS models (Realizable k-ε, Standard k-ε, k-ω, SST) — convergence behavior and PIV agreement side by side.

Mesh Independence

See why a 20-mesh basket needs 3.5× more cells than a 10-mesh, with quantitative convergence metrics.

CFD vs PIV Validation

Validation summary across 7 horizontal measurement planes, plus flow rate comparison from published data.

Power Number & Mixing Time

Power consumption (Po) comparisons across basket configurations, and blend time data at different fill volumes.

CFD Knowledge Base

Expert-level setup guidance: turbulence model, mesh sizing, solver settings, boundary conditions — all referenced to published results.

Key Finding

Of 4 RANS turbulence models tested, only Realizable k-ε with Standard Wall Functions achieved both convergence (residuals < 10−4) and agreement with PIV experiments at all measurement locations.

Conditions: USP Apparatus 1 per USP <711>, 100 RPM, 900 mL water at 20 °C, Re ≈ 1075. Basket mesh sizes: 10- and 20-mesh (CFD), 10-, 20-, and 40-mesh (PIV).

Run It Locally

# Clone and run
git clone https://github.com/Chadakarn23/usp1-hydrodynamics-research.git
cd usp1-hydrodynamics-research
pip install -r requirements.txt
streamlit run app.py

Requires Python 3.10+. Opens at http://localhost:8501.
